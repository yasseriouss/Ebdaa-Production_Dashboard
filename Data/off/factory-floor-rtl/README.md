# Wood work orders dashboard (مصنع الأخشاب)

Next.js App Router UI for active wood work orders: Kanban/table dashboard, RTL Arabic layout, Shadcn-style components, and local persistence for supervisor stage edits.

## Run locally

```bash
cd Data/off/factory-floor-rtl
npm install
npm run dev
```

From the monorepo root (after `pnpm install`):

```bash
pnpm dev:wood-dashboard
```

## Data

`src/data/wood_work_orders.json` is a copy of `Data/wood_work_orders.json`. Refresh that file when the source export changes.

## Package name

npm package name: `factory-wood-dashboard` (folder: `factory-floor-rtl`).
