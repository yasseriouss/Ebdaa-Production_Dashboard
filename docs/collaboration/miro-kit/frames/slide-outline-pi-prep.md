## Frame — اعتمادات هندسية

- جداول المواءمة موجودة تحت **`tables/`** لاستخدامها كنقطة انطلاق (استيراد بيانات لوحة أو `table_sync_rows` عبر MCP مستقبلاً).

## Frame — مخاطرة متابعة واحدة هذا الـ Increment

عندما يكون **السياسة** جاهزة: إضافة حيث SQL في خدمات `wooden.service`/`metal.service`/`factoryHub.service` وفق نفس نمط `dataScopeFilter` الموجود اليوم على السعة/الموظفين؛ عدم الانطلاق بتخمين عمود لم يعتمد قط أعمالياً.

## Frame — الهدف المقاييس الشفوي

لوحة واحدة قبل نهاية الـ PI تُبيِّن تطابق مخطّط **نطاق الصفوف** قبل/بعد للحسّاسية للإنتاج؛ حتى قبل كتابة الكود اعتمد مخطّطاً واحداً مشتركًا في Miro مقابل هذا المجلّد.

## Frame — الموازي «لا تعطل الأساسى»

- `local-routing-translator-followup` ومزامنة نص Ebdaa — خلف قائمة ظهرية؛ لا تأخذ مقدّمة السعة من فريق الأساس قبل إغلاق النطاق.
