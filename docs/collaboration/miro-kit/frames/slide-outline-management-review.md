# إطار: Management Review Meeting (محتوى لـ Miro)

## Frame — ملخص تنفيذي لمدة 15 دقيقة

- الوضع مقابل خطة الطريق: المراحلة أ–هـ؛ **الباقي الفعلي** `perm-scope-queries-production-hub`.
- خطر اليوم: وصول ذو دور محدّد قد يُشاهِد قائمة أوامر إنتاج شاملة **من دون تصفية نطاق مصنع/قسم على مستوى الصف**؛ الحماية الحالية هي **مسار + RBAC**.
- قرار مقترَح أمام الإدارة: توقيت وتشكيل قرار حقول المصنع/القسم قبل التنفيذ التقني.

## Frame — تكامل ومخاطرة بيانات

- مصدر المرجع (hub JSON) وجداول «معيارية» خشب/معدن — المرجع الوثائقي `docs/legacy-adapter-factory-hub.md`.
- متابعة **محوّل routing ثمان مراحل مقابل أربعة مراحل wooden** من دون قطع المتصلين؛ خلف الخطة كلها.

## Frame — جاهزية العرض أمام الزملاء

- يجب تجربة مستخدم تجريبي + بيانات seed (`POST /factory-hub/seed`) قبل الاجتماع.
- تأكّد أي واجهة تعرض اليوم **`apps/web`** أم **`artifacts/factory-app`** واذكر ذلك في الانطلاق لعدم خلط المتوقّع مع README الحالي.
